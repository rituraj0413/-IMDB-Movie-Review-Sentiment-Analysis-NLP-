# IMDB Movie Review Sentiment Analysis (NLP)

This project predicts whether a movie review is **Positive** or **Negative**
using a **Simple RNN model** trained on the IMDB dataset.

## Tech Stack
- Python
- TensorFlow / Keras
- Streamlit
- NLP

## Features
- Text preprocessing
- Sentiment prediction
- Interactive Streamlit UI

## How to Run
```bash
pip install -r requirements.txt
streamlit run main.py
